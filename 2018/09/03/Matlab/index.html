Matlab 101

title: Matlab 101
author: yuan

Lecture 1

  • 以矩阵为基本单位，无需预先指定矩阵维数（动态定维）
  • Matlab和Maple相结合
  • 使用工具箱
  • 不要用 i, j 做变量名
  • eps:函数，可以得到浮点数的相对精度，也就是相邻两数之间的最小距离。

eps(le20) , ans=16384 % As a result

(le20+10)-le20 = 0

  • 用eps代替0，可以避免0作分母的情况
      function y = mysin(x)
  % 实现与系统函数sinc 同样的功能 
  % SINC sin(pi*x)/(pi*x) 
  indexI = find(x==0);  
  x(indexI)=eps(0)+x(indexI);
  y = sin(pi*x)./(pi*x)
  • 分号和续行符号…
  • 字符串，二维字符串的时候，短的要用字符补齐。

  • 创建结构数组，可以用struct函数预分配一个结构数组
    str_array = struct('field1',val1,'field2',val2)

  • 搞清楚. 和 ，./和 /的区别。

  • 转置符号 ‘ 涉及到复数要小心，因为这个符号其实是共轭转置。
  • 而 .’ 才是转置，尽量用transpose(),ctranspose.%conjugate transpose
  • 脚本式的m文件和函数式的m文件。
